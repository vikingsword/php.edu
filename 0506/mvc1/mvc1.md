# mvc1 极简版

- `index.php`: 这就是一个页面,也就是视图层
- mvc 是对这个视图的解读

## M: model

- Model: 模型层
- 功能: 数据来源与常用操作
- 其实就是页面呈现给用户的数据
- 数据来源可以是文件,也可以是数据表

## V: View

- View: 视图层
- 功能: 呈现 Model 模型层数据,其实就是模板
- 数据呈现的方式,本例用的是表格,格式化显示
- 其实对于用户来说, 感兴趣的只有视图,其它不关心或不可见

## C: Controller

- Controller: 控制器层
- 功能: 根据模型层数据,选择适合的视图进行展现
- 控制器是系统功能集中地, 程序员重点关注和工作目标
